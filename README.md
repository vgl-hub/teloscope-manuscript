# Teloscope-manuscript
Scripts and data used for the VGP telomeres manuscript

## Code
Including code to process Teloscope outputs, parsing BED/BEDgraphs and reports.
This also includes code for exploratory analyses and to generate plots. 

### 1. Processing
This folder includes scripts to process raw -> processed data.

### 2. Analysis
This folder operates with processed data to generate dataframes, statistics and summaries.

### 3. Plotting
This folder operates with processed data to generate exploratory and final plots.

## Data
### 1. Raw
This includes heavy unparsed genomics data. 

### 2. Processed
This includes parsed Teloscope outputs and condesned dataframes related to the latest VGP data freeze available.

### 3. Other
This includes data generated by collaborators. 

## Plots
### 1. Draft
First unpolished figures to explore biological questions.

### 2. Final
Figure panels included in the manuscript preprint and Teloscope repo.
